# Expense Tracker using Python 
A simple command-line based Monthly Expense Tracker built using Python.
This project helps users log daily expenses, categorize spending, monitor monthly budgets, and export all records to a CSV file for future reference.

## Features:
1. Add expenses with date validation (YYYY-MM-DD)
2. Categorize expenses (Food, Travel, etc.)
3. Track total monthly spending
4. Get warnings when budget is exceeded
5. Save all expense records into a CSV file
6. View previously added expenses

## How to Run:
1. Clone the repository:
git clone https://github.com/hrithikaulimiri-boop/Expense-Tracker.git

2. Navigate into the project folder:
cd Expense-Tracker

3. Run the program:
python tracker.py

## Menu Options:
* Add Expense – Log new expenses with date, category, amount and description
* View Expenses – Display all entered expenses
* Track Budget – Compare expenses with your monthly budget
* Save Expenses – Export all data to a CSV file
* Exit – Close the application

## Technologies Used:
* Python
* File Handling
* CSV Module
* Dictionaries & Lists
